# pnpm-update-script

Updating the node version being used in all global packages after updating pnpm

## Project Setup

```sh
pnpm install
```

### Run src/index.ts

```sh
pnpm start
```

### Build the project with Rollup

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
